================================================================
🆘 TROUBLESHOOTING
================================================================

Problem: Tools not showing in Claude?
Solution:
   1. Close Claude completely (check System Tray)
   2. Kill Claude.exe in Task Manager
   3. Verify JSON syntax
   4. Restart Claude

Problem: Port already in use?
Solution:
   Use different port:
   npx brave-real-browser-mcp-server@latest --mode http --port 8080

Problem: Brave Browser not found?
Solution:
   Add BRAVE_PATH in config:
   "env": {
     "BRAVE_PATH": "C:\\Program Files\\BraveSoftware\\Brave-Browser\\Application\\brave.exe"
   }
