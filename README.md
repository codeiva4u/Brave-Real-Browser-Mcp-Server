## üé® IDE Configurations

### 1. Antigravity AI IDE / Gemini 3 Pro
Add to your config:

```json
{

    "brave-real-browser": {
      "command": "npx",
      "args": ["-y", "brave-real-browser-mcp-server@latest"],
      "env": {

    }
  }
}
```
